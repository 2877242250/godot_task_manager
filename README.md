# godot_task_manager
可以导出CSV表格。
这是我的第一个开源项目，基于Godot4游戏引擎（beta4）。
注意**这不是一个完整的项目**。某些功能缺失，可以根据自己的要求进行修改。

# 内容
- 客户端程序：
./taskManager
- 服务器程序：
./taskManager_server
# 操作环境
Godot4beta4
# 简介
- TaskManager有两个用户级别：管理者和操作员
管理者可以管理项目的内容，操作员可以向项目添加记录
服务器端窗口可以注册帐户、添加项目和导出项目的CSV表
